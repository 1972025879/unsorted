# np
## np.random.rand
```bash
np.random.rand()          # 生成一个 [0,1) 之间的随机数  
np.random.rand(3)         # 生成形状为 (3,) 的随机数组  
np.random.rand(2, 3)      # 生成形状为 (2,3) 的随机数组
:'
假设原始 gt 是一个圆形：
如果 iou_target = 0.95：生成的 seg 可能只是边界略微变形
如果 iou_target = 0.85：生成的 seg 可能边界有较明显的变形或缺口
如果 iou_target = 0.98：生成的 seg 与 gt 几乎完全一样
'
```
